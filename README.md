# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/tusharaitechie/python-for-ds-ml.git
```

Move into the project directory:

```bash
cd python-for-ds-ml
```

Install the required libraries:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn requests beautifulsoup4 xgboost jupyter
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
